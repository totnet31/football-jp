# 運用ルール・手動投稿ガイド

---

## 自動 vs 手動の役割分担

| 役割 | 担当 | 内容 |
|------|------|------|
| **情報提供** | 自動（bot） | 試合スケジュール・結果・ゴール速報・週次まとめ |
| **感情・意見** | 手動 | 戦術コメント・感想・選手の素晴らしさを語る |
| **ストーリー** | 手動 | 選手キャリア・記録達成・移籍ニュース |
| **コミュニティ** | 手動 | リプライ・引用RT・アンケート |
| **危機対応** | 手動 | 誤投稿訂正・炎上回避 |

### 鉄則：botは「情報」、人間は「温かみ」

自動投稿だけのアカウントは機械的で飽きられる。手動投稿で「人が運営している感」を出すことで、フォロワーとの関係性が生まれる。

## 手動投稿のテーマ案

### カテゴリ1：戦術・分析（週1回目安）
- 「〇〇選手の今節のプレーがすごかった」
- 「このフォーメーションで〇〇が輝く理由」
- 「クラブの戦術変更で出場機会はどう変わる？」

**例：**
```
久保建英がレアル・ソシエダで輝く理由

攻撃の自由度が高く、ポジションに縛られない戦術が
彼の持ち味である「ドリブル×パス×シュート」を
最大限に引き出している

今節のプレーは特に良かった⚽
football-jp.com でも追跡中
#久保建英 #ラリーガ
```

### カテゴリ2：選手キャリアの節目（その都度）
- 公式移籍発表
- 代表選出
- ゴール・アシストの通算記録達成
- 怪我からの復帰

**例：**
```
おかえり！{選手名}、{N}か月ぶりの復帰戦

怪我での長期離脱は辛かったと思います
まずは無理せず、またピッチで輝く姿を見せてください

応援しています⚽
#{選手名} #海外サッカー
```

### カテゴリ3：ファンとの対話（週2〜3回目安）
- 「今週一番気になった選手は？」（アンケート活用）
- 「〇〇選手の次節予想」
- 「W杯出場してほしい選手は？」

**例：**
```
質問です！

今シーズン「一番活躍している」と思う日本人海外選手は誰ですか？

🔁 久保建英
❤️ 遠藤航
💬 その他（コメントで！）

#海外サッカー #日本人選手
```

### カテゴリ4：サイト紹介・機能告知（月2〜3回）
- 新機能追加時
- W杯特集ページ更新時
- アフィリエイトリンク（自然な形で）

## 投稿時間ベストプラクティス

| 時間帯 | 理由 | 投稿内容 |
|--------|------|----------|
| **朝7:30** | 出勤前・通学前にスマホチェックが多い | 今日のスケジュール（bot） |
| **昼12:00** | 昼休みのSNSチェック | 試合結果・軽い情報 |
| **夜21:00〜22:00** | 最もエンゲージメントが高い時間帯 | 手動投稿・試合結果・感想 |
| **土日午後** | 試合が多く、ファンが見ている | 試合中・試合後の投稿 |

### 避けるべき時間帯
- 深夜0〜5時（リーチが極めて低い）
- 平日昼14〜16時（最も閑散）

## リプライ戦略

### 対象アカウント（リプライ・引用RT）

**優先度高：**
- 日本人選手の所属クラブ公式アカウント
- 日本サッカー協会（@jfa_samuraiblue 等）
- DAZN Japan（@DAZN_JPN）
- スポーツ系メディア（Goal Japan、フットボールチャンネル等）

**優先度中：**
- サッカー系インフルエンサー（フォロワー5,000〜50,000程度）
- サッカー好きの一般ユーザー（バズっている投稿へのリプライ）

### リプライの作法
- **批判・論争はしない**（特定選手・監督・クラブへの批判はNG）
- 情報提供型でリプライする（「〇〇選手の今週の試合情報はこちらでも確認できます」等）
- 感情的なリプライはしない（冷静・中立を保つ）

### 引用RTの活用
- 公式クラブが選手のゴール動画を投稿 → サイトリンク付きで引用RT
- スポーツメディアが選手記事を投稿 → コメント付きで引用RT

## 危機管理

### ケース1：誤った情報を投稿してしまった場合

**手順：**
1. 誤投稿をすぐに削除
2. 正しい情報で訂正投稿
3. 訂正ツイートのフォーマット：

```
【訂正】先ほどの投稿について

× 誤：{誤った情報}
○ 正：{正しい情報}

ご指摘ありがとうございました。
正確な情報をお届けするよう努めます。
```

4. 原因を調べてデータソースを修正

### ケース2：炎上・批判が来た場合

**基本方針：** 無視 or 丁寧に事実確認のみ
- 感情的な言い返しは絶対しない
- 明らかな誤情報指摘には感謝して訂正
- 悪意ある攻撃はスルー・ブロックも検討

### ケース3：著作権関連クレームが来た場合

- 当該ツイートを即削除
- 謝罪は最小限（非を認めすぎない）
- 再発防止措置を取る

## 著作権配慮ルール

### してはいけないこと（絶対NG）
- 選手・試合の写真を無断でツイートに貼る
- 公式放送の映像・静止画のスクリーンショット投稿
- 試合実況のコピペ投稿
- 選手の公式SNS画像の無断引用

### してよいこと（基本的にOK）
- テキストのみの試合情報投稿
- 自分で作ったテキスト画像（スコア・選手名等を自分でデザインしたもの）
- 公式クラブ・選手アカウントの投稿を「引用RT」する（元ツイートを参照する形）
- 選手名・クラブ名・スコアなどの事実情報の記述

### グレーゾーン（慎重に）
- クラブの公式ロゴ（小さく・情報の一部として使用するのはグレー）
- 選手アイコン画像（ウィキメディアコモンズのCC画像のみ使用可）
