## 7. ハッシュタグ運用ルール

### 必須ハッシュタグ
| 用途 | ハッシュタグ |
|------|-------------|
| 全投稿に共通 | `#海外サッカー` |
| 選手関連 | `#日本人選手` |
| W杯関連 | `#W杯2026` `#日本代表` |

### リーグ別ハッシュタグ
| リーグ | ハッシュタグ |
|--------|-------------|
| プレミアリーグ | `#プレミアリーグ` `#PremierLeague` |
| ブンデスリーガ | `#ブンデスリーガ` `#Bundesliga` |
| ラ・リーガ | `#ラリーガ` `#LaLiga` |
| セリエA | `#セリエA` `#SerieA` |
| リーグ・アン | `#リーグアン` `#Ligue1` |
| エールディビジ | `#エールディビジ` |
| ベルギーリーグ | `#ベルギーリーグ` |

### 選手名ハッシュタグ
- 日本語のみ（例：`#久保建英` `#遠藤航` `#堂安律`）
- 1投稿で選手タグは最大2名まで（ごちゃつき防止）

### ハッシュタグ数の上限
- 1投稿：最大4タグ（それ以上はリーチが下がる傾向）
- 構成例：`#海外サッカー` `#{リーグ名}` `#{選手名}` `#日本人選手`
