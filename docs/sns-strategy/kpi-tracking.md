## UTM設計（GA4連携）

Xからの流入を正確に把握するため、URLにUTMパラメータを付与する。

### UTMパラメータ設計

| パラメータ | 値 | 例 |
|-----------|----|----|
| utm_source | twitter | - |
| utm_medium | social | - |
| utm_campaign | 種別コード | 以下参照 |
| utm_content | 投稿種別 | 以下参照 |

### キャンペーンコード一覧

| 投稿種別 | utm_campaign | utm_content |
|----------|-------------|-------------|
| 朝スケジュール（bot） | bot_morning | schedule |
| 試合結果（bot） | bot_result | result |
| ゴール速報（bot） | bot_goal | goal |
| 週次まとめ（bot） | bot_weekly | weekly |
| 手動：選手紹介 | manual_player | player_feature |
| 手動：戦術コメント | manual_tactic | tactic |
| 手動：W杯特集 | manual_wc2026 | worldcup |
| ピン留めツイート | pinned | top |

### UTM付きURLサンプル
```
https://football-jp.com/?utm_source=twitter&utm_medium=social&utm_campaign=bot_morning&utm_content=schedule

https://football-jp.com/players/kubo-takefusa/?utm_source=twitter&utm_medium=social&utm_campaign=manual_player&utm_content=player_feature
```

## GA4 設定

### 必要なイベント設定

| イベント名 | トリガー | 目的 |
|-----------|---------|------|
| `page_view` | 全ページ表示（デフォルト） | 流入計測 |
| `session_start` | セッション開始（デフォルト） | ユニーク訪問者数 |
| `click` リンク | アフィリエイトリンククリック | 収益化計測 |

### 確認すべきGA4レポート
1. **集客 → トラフィック獲得** → ソース/メディア別で `twitter / social` を確認
2. **集客 → UTMキャンペーン** → 投稿種別ごとの流入数確認
3. **エンゲージメント → ランディングページ** → Xからの流入がどのページに来ているか

## Search Console 連携（指名検索計測）

### 計測目的
「football-jp」「日本人 海外サッカー」等の指名・関連検索の増加を追跡する。

### 確認すべき指標
| クエリ | 確認内容 |
|--------|---------|
| `football-jp` / `football jp` | ブランド認知の直接指標 |
| `日本人 海外サッカー 選手` | カテゴリ検索での露出 |
| `{選手名} 試合` | 個別選手流入 |

### 月次レビューチェック
- 指名検索数の前月比変化
- 検索順位の推移（主要クエリ上位10位以内かどうか）

## アカウント成長フェーズの判断基準

### フェーズ1：立ち上げ期（フォロワー0〜500人）
**特徴：** フォローバック率が高い。インプレッションは低い。
**施策：**
- とにかく継続投稿（毎日最低1投稿）
- 関連ハッシュタグで検索して積極リプライ
- サッカー系アカウントをフォロー

**移行判断：** フォロワー500人達成 かつ 月間インプレッション50,000超

---

### フェーズ2：成長期（フォロワー500〜3,000人）
**特徴：** フォロワーの連鎖フォローが始まる。バズ投稿が出始める。
**施策：**
- エンゲージメント率の高いコンテンツを分析・増量
- スレッド投稿の活用（情報量増加）
- インフルエンサーとの自然なリプライ連携

**移行判断：** フォロワー3,000人達成 かつ 月間インプレッション300,000超

---

### フェーズ3：安定期（フォロワー3,000〜10,000人）
**特徴：** オーガニックフォローが安定。コミュニティが形成される。
**施策：**
- W杯2026に向けた専門コンテンツ強化
- メディア・クラブとのコラボ検討
- アフィリエイト収益の最大化

**移行判断：** フォロワー10,000人達成
