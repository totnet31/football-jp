# SNS戦略 全体像

football-jp.com の認知拡大・流入増加を目的とした X（旧Twitter）一本集中戦略。

---

## ビジネスゴール

| 目標 | 内容 |
|------|------|
| **認知拡大** | 「日本人選手の海外試合情報といえばfootball-jp.com」のブランドポジション確立 |
| **サイト流入** | Xからの月間セッション数を3か月で1,000→5,000に伸ばす |
| **収益化基盤** | DAZN・U-NEXT等アフィリエイトへの送客数を増やす |
| **コミュニティ** | サッカーファンとの双方向コミュニティ形成 |

---

## KPI サマリ

| 指標 | 1か月後 | 3か月後 | 6か月後 | 1年後 |
|------|---------|---------|---------|-------|

| 月間インプレッション | 50,000 | 200,000 | 500,000 | 1,500,000 |
| Xからの月間セッション | 300 | 1,500 | 5,000 | 20,000 |

## ロードマップ

### フェーズ1：基盤構築（1〜2か月目）

**やること**
- Xアカウント開設・プロフィール整備
- 自動投稿bot セットアップ（朝のスケジュール投稿のみ）
- 手動投稿を週3〜5回で習慣化
- ハッシュタグ戦略の確立

**目標**
- フォロワー500人
- 毎日1〜2投稿を安定稼働

---

### フェーズ2：拡大（3〜4か月目）

**やること**
- 自動投稿を試合結果・週次まとめに拡張
- 投稿画像の動的生成（選手名・試合結果をテキスト画像化）
- 公式クラブ・選手アカウントへのリプライ戦略開始
- インフルエンサーとの緩やかな連携

**目標**
- フォロワー1,500人
- Xからの月間セッション1,500

---

### フェーズ3：深化（5〜6か月目）

**やること**
- W杯2026特化コンテンツ強化（カウントダウン・組み合わせ抽選等）
- スレッド形式の戦術分析投稿
- フォロワーへのアンケート・エンゲージメント投稿
- GA4×UTMによる流入分析の本格運用

**目標**
- フォロワー5,000人
- Xからの月間セッション5,000

---

### フェーズ4：スケール（7〜12か月目）

**やること**
- 人気コンテンツの分析・最適化
- 試合ハイライト動画クリップへの言及（著作権範囲内）
- 他メディア・ブロガーとのコラボ
- 必要に応じてInstagram・YouTubeへの横展開検討

**目標**
- フォロワー15,000人
- Xからの月間セッション20,000

---

## 戦略の核心3点

1. **データ自動投稿 × 手動の温かみ**
   - botが情報提供（速報・スケジュール）、人間が感情・意見・ストーリーを語る
   - 「情報サイト」ではなく「サッカーファンの声」として認知される

2. **W杯2026を最大の追い風に**
   - 2026年W杯まで約1年（2026年6月〜）
   - 日本代表・海外選手の注目が高まる時期に合わせてコンテンツを集中投下

3. **無料プラン内で賢く運用**
   - X API無料プラン：月1,500投稿まで
   - 1日平均3投稿（90/月）＋自動5投稿（150/月）＝月間約240投稿で十分に収まる
